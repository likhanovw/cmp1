Telegram-бот виртуальной валюты
================================

Этот проект реализует Telegram-бота виртуальной валюты по следующему ТЗ:

- Баланс и история операций
- Переводы между пользователями
- Запрос средств через QR-код и deeplink
- Админ-панель для начислений и списаний

## Стек

- Python 3.11+
- aiogram 3.x — Telegram Bot API
- SQLite — хранение данных (пользователи, балансы, транзакции, запросы платежей)
- qrcode[pil] — генерация QR-кодов

## Быстрый старт

1. Создайте и активируйте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Установите переменные окружения:

```bash
export BOT_TOKEN="ВАШ_TELEGRAM_BOT_TOKEN"
export SUPER_ADMIN_ID="123456789"  # ваш Telegram user id
```

4. Запустите бота:

```bash
python bot.py
```

## Примечания

- По умолчанию используется SQLite-файл `bot.db` в корне проекта.
- QR-коды генерируются как PNG-файлы и отправляются пользователю напрямую.
